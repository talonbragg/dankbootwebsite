About

This is a bot that allows you to Ban Kick and Purge. If you don't know what those commands usually mean on discord, google it, but purge means to delete messages. More moderation commands will be coming out soon, but for now, sit tight, relax, and enjoy some !!weather. (That was a hint).

Command Help

!!help

This displays the help section

!!kick

Command usage: !!kick <@username> <reason>. This command allows you to kick people. It requires thenrole of Admin.

!!ban

The usage of this command is the same as !!kick and also requires the Admin role.

!!purge

This command also requires the role of Admin. Command usage: !!purge <number of messages to delete>

!!weather Command usage: !!weather <town>

This displays the weather of the town that you chose in the command.

!!mute Command usage: !!mute <@username>

!!unmute Command usage: !!unmute <@username>

Note: This command reqires a role named mute and I recommend that you make it so that the mute role is not able to send messages.

!!shortenlink Command usage: !!shortenlink <link>

This command allows you to shorten any web link.
